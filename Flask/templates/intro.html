Give the shape and get the formula!!!!



Smart Maths tutor system is a web based graphical user interface where a user gets to draw shapes of mathematical figures such as square, triangle, circle etc. for which the output would be related formulas to the drawn figure. The use of interactive technology in learning for these students is as natural as using a pencil and paper were to past generations.
